# 🚀 A股强势股捕捉器

## 功能

基于 **N字型态** 经典技术形态，实时筛选强势股。

### N字型态特征

**第一笔（上涨）**：
- 放量突破前期高点
- 成交量明显放大

**第二笔（回撤）**：
- 回撤不破突破点
- 缩量回调（量能萎缩）

**第三笔（再涨）**：
- 再次放量上涨
- 突破第一笔高点

## 输入

- 市场：A股（默认）
- 形态类型：N字型态

## 输出

- 推荐股票 3-5 只
- 每只包含：
  - 代码 + 名称
  - 所处板块
  - N字型态阶段（突破/回撤/再次启动）
  - 强势度 ⭐⭐⭐/⭐⭐/⭐
  - 放量情况
  - 止损位 + 目标位
  - 买入理由

## 结合分析

- 结合当天资金热度（板块主力净流入）
- 结合板块联动效应

## 数据源

- 东方财富 / 同花顺
- 备用：新浪财经 / 腾讯财经
