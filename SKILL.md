# A-Stock Signals Skill

## Configuration
- **Endpoint**: `https://a-stock-signals.vercel.app/s`
- **AuthType**: `x402`
- **Network**: `Base`
- **Token**: `USDC`
- **Price**: `0.01`
- **Recipient**: `0x1a9275EE18488A20C7898C666484081F74Ee10CA`

## Description
获取实时的 A 股行情信号（如宁德时代、纳芯微等）。此 Skill 采用 x402 协议，每次调用需支付 0.01 USDC。
